## Known Gaps

- Live provider smoke tests are implemented behind `--live`; they were not run in this pass because live calls can spend API credits or require external model access.
- `--local` now routes to a wired WhisperX adapter, but real local execution still depends on installing `dubsync[local]` plus WhisperX runtime dependencies.
- Live pyannote overlap detection was not smoke-tested; it requires `dubsync[diarize-local]`, accepted model terms, and a Hugging Face token or local model path.
- MMS forced alignment is wired through the optional `ctc-forced-aligner` Python API, but real model execution was not smoke-tested in this workspace.
- Deterministic energy/fixture VAD is wired; neural/model VAD remains a future optional upgrade.
- Live LLM speaker-character mapping is implemented through the configured LLM adapter, but it was not smoke-tested against real provider responses in this workspace.
- Live LLM token cost metering has deterministic provider-response tests, but it was not smoke-tested against real Gemini/OpenAI/Anthropic responses in this workspace.
- Gemini audio-snippet double-checks are implemented with deterministic extraction and fake-SDK coverage, but were not smoke-tested against the real Gemini API in this workspace.
- OpenAI and Anthropic token prices require explicit config overrides until model-specific billing defaults are confirmed.
- The folder is not a git repository, so TDD checkpoint commits could not be created.
